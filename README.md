# ForegroundSeg

AI-powered foreground/background segmentation on any uploaded image using PyTorch.

## Features

- DeepLabV3
- Automatic segmentation
- Overlay visualization
- Download foreground
- Download background

## Technologies

- Python
- PyTorch
- TorchVision
- Streamlit

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
